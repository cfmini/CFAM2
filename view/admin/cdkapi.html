CDK兑换接口
通过API发送CDK

返回格式： JSON

请求方式： GET/POST

接口地址： {$domain}/index/AjaxSendCdk

请求参数说明：

名称 必填 说明 示例
user 是 用户账号 uid=用户账号
cdk 是 cdk代码 cdk=cdk代码

响应结果说明：

名称 类型 必填 说明
code number 是 状态码：200成功 / 500失败
msg String 是 状态对应描述

请求示例：

{$domain}/index/AjaxSendCdk?user=test111&cdk=nYvDXykd7i19czPJJoTvkrZblVjFas8o

返回示例：

{
    "code": 200,
    "msg": "发送成功",
}
